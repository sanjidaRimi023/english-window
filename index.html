Frequently Asked Questions

what is the difference between var, let, and const?

Answer : The difference between var, let and const is-

  • var => Var declares variable with function or global scope and allows re-declaration and update eithin the same scpoe
  • let => it's declares variable with block scope, allowing updates but not re-declaration with in same block.
  • const => it's declares block-scoped variables that can't be change or reassigned after their initial assignment
    • What is the difference between map(), forEach(), and filter()?

    Answer : The difference between map(), forEach() and filter() is-

  • map()=> Map create a new array from calling a function for every array element. it's dosen't execute the function for empty elements.

  • forEach()=> The main purpose of forEach method is it executes a provided function once for each array element. In other words, iterates over the array. Often used in functional programming. Always returns undefined.

  • filter()=>The filter() method creates a new array filled with all the elements of the old array that pass a certain test, provided as a function.It can be thought as of a loop, that is specifically for filtering in/out certain values from an array.

They offer several benefits over traditional function expressions, including an implicit "this" binding, implicit return, and a more concise syntax. By using arrow functions, we can write cleaner and more efficient code, making our JavaScript code easier to read and maintain.But we can't use "arguments" and "new" keyword in array function like normal function.

    how JavaScript Promises work

    Answer : Promises are a fundamental part of modern JavaScript and are widely used in APIs like fetch, timers, and other asynchronous operations.They have three states: pending, fulfilled, and rejected.

    how closures work in JavaScript

    Answer : A closure is a function that retains access to its lexical scope, even after the outer function has finished executing.It's created whenever a function is defined inside another function.
